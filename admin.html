🤖 Auto-Fetch Matches
Automatically fetch today's matches and scan for live streams

This feature uses free sports APIs to get today's match schedule, then scans known streaming sites to automatically find and add matches with working stream links.

Or fetch matches only (no scanning)

How it works:

  1. Uses TheSportsDB free API to get today's match schedule
  2. Scans streaming sites for available .m3u8 links
  3. Matches team names with found streams
  4. Auto-adds matches with streams to your Firestore database
